1. Go to 
https://plutocode.ph.unito.it/
and download the PLUTO tar.gz file

2. Make a directory in which you want to install PLUTO.
Copy PLUTO tar.gz file in that directory

3. Open the terminal and run
gunzip  pluto-xx.tar.gz
tar xvf pluto-xx.tar
this will create the folder PLUTO/ in the specified directory.

4. Set the environment variable PLUTO_DIR to point to your code directory. For that, open the .bashrc file and write
export PLUTO_DIR=<path to PLUTO directory>
